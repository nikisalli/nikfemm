# nikfemm
a 2d physics simulator written in C++ from scratch

# how to test it
clone the repo first:

``` git clone https://github.com/nikisalli/nikfemm.git ```

create build dir

``` mkdir build && cd build ```

configure and build

``` cmake .. ```

``` make -j4 ```

run an example

``` ./bin/test ```
